# Tutorial for using VuePress with GitHubpage to make blog

Recently, some friend in freecodecamp Hanoi group who start write blog by using github page. It make me curious about making a blog by githubpage. I start read guide in tutorial in home page to create static page in github page. Every thing is seem easy by creating a index.html file with several the tags. But no blog like this, I need decorate my blog and I continue find out how to add theme for my blog. So I found Jekyll that support for github page. Great! the themes of Jekyll arr really nice but if I use it, I need install ruby and setting something to apply this theme. I don't want install a something that I hardly use. So I have working experience with Vuejs and I think I should make my blog by Vuejs.

VuePress is new production of EvanYou that to generate the static web and is SEO-friendly. You can read about the main feature of VuePress in here. I feel the document of Vuepress is not good and I must take more time to understand how to use.

# Install and run Vuepress

So I start install by npm:

$ mkdir blog
$ cd blog
$ npm init
$ npm install --save vuepress

beacuse Vuepress don't have CLI so you must config package.json command to run dev mode and build mode.

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

So I try open dev mode

$ npm run docs:dev

and my terminal print message

  VuePress dev server listening at http://localhost:8080/
  [18:50:15] âœ” successfully compiled.

If you run http://localhost:8080/ you will see page 404 not found, beacause you don't specify the content for home page. In Vuepress, README.md is convert index.html in each folder. So you must create file README.md with content and try again.

Then I try run build mode

$ npm run docs:build
the production will be created in .vuepress folder (a hidden folder) to explore it you just run
$ cd .vuepress

That is all step you need to build static web by Vuepress. We continue apply our static web with github page.
